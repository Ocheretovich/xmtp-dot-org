📘 **Universal Allow/Block Prefs**:

XIP-42 4 establishes ‘allow’ and ‘block’ permission preferences, enabling users to explicitly specify which contacts should be able to reach them and which should be blocked across all inbox apps.

**Ideas**:

- Subscriptions: Opt into communications with a single click.
- Broadcasting: Send messages to multiple recipients.
- Request Inbox: Have full control over who can message you.

**Resources**:

- [Understanding user consent](/docs/build/user-consent)
- [Spam documentation](/docs/build/spam)
- [0xDesigner post](https://warpcast.com/0xdesigner/0x52fa0e7d)
